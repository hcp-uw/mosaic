Module 1 is the user experience way to startup and shutdown mosaic.
Module 2 is for developer testing mosaic

Make sure Go is installed on your computer.

As of 2/3/2026 The log file doesnt seem to work although everything else should be good.

## Module 1:
While in mosaic:
.\install.ps1

To shutdown:
- mos shutdown
or
- .\install.ps1 -s

## Module 2:

Developer tools:
- ./install.sh -d -> extra debugging info including PID
